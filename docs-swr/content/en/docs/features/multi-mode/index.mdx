# Multi-mode Support

Multi-mode support is a powerful feature of Variables Xporter that enables you to easily export multi-mode design variables, such as light/dark themes, brand themes, and more.

## What is Multi-mode?

Multi-mode allows a variable to have different values under different conditions. Common modes include:

- Light/Dark themes
- Brand themes
- Responsive breakpoints
- Layout density

Variables Xporter supports **compound multi-mode variables**. For example, the `colors/primary/DEFAULT` variable can be affected by both dark/light themes and brand themes simultaneously, resulting in the following output:

## How It Works
Variables Xporter traces all referenced variables in the selected export collection until it finds the original values. This means you can reference variables across collections, and Variables Xporter will fully reconstruct the reference chain.
