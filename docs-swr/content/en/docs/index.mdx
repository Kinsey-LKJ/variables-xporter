# Variables Xporter

Variables Xporter is a Figma plugin that exports Figma variables to Tailwind CSS configuration files or CSS variables, making it effortless to sync your design system with your development environment!

## Export Formats
- **Tailwind CSS**: Generates configuration files and corresponding CSS files
- **CSS Variables**: Generates standard CSS custom properties
