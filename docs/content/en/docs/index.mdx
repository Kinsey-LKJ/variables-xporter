# Variables Xporter

Variables Xporter is a Figma plugin that exports Figma variables as Tailwind CSS configuration files or CSS variables. It makes it easy to synchronize design systems with the development environment!

## Export formats

- **Tailwind CSS**: generates configuration files and corresponding CSS files
- **CSS variables**: generates standard CSS variables

## Figma variable organization suggestions

We provide comprehensive Figma variable organization suggestions to help you better manage variables and ensure the usability of the export results.
