# 间距系统

间距和尺寸是设计系统中的关键元素，有助于我们创建一致的视觉层次和布局节奏。让我们来看看如何组织这些重要的设计变量。

## 顶级目录

请使用 `spacing` 作为所有间距和尺寸变量的顶级目录。以下是一些常见的命名示例：

语义化命名：
- `spacing/sm` - 小间距，适用于紧凑的元素布局
- `spacing/md` - 中间距，适用于常规的元素间距
- `spacing/lg` - 大间距，适用于区块分隔
- `spacing/xl` - 超大间距，适用于主要区域分隔

数字化命名（原子变量）：
- `spacing/0` - 无间距（0px）
- `spacing/1` - 最小间距（通常为 4px）
- `spacing/2` - 双倍最小间距（通常为 8px）
- `spacing/4` - 四倍最小间距（通常为 16px）

## 单位转换

Variables Xporter 提供了智能单位转换功能。您可以在 Figma 中使用更直观的 px 单位进行设计，然后在导出时选择“使用 Rem 单位”选项。工具会自动将其转换为无障碍友好的 rem 单位。
