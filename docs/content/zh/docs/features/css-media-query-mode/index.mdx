# CSS 媒体查询模式

CSS 媒体查询模式让你能够为不同的视口尺寸或设备特性定义变量值，实现响应式设计。

## 工作原理

Variables Xporter 支持在变量模式名中使用标准的媒体查询语法：

变量模式命名规则：
**`{media-feature}:{value}`**

例如：
`min-width:768px`、`max-width:1024px`

### 支持的媒体特性

- `min-width`
- `max-width`
- `min-height`
- `max-height`
- `orientation`
- `aspect-ratio`
- `min-aspect-ratio`
- `max-aspect-ratio`
- `prefers-color-scheme`
- `prefers-reduced-motion`
- `display-mode`

## 使用示例

### 1. 响应式间距

在 Figma 中设置不同断点的容器宽度：

```                                          
 ┌────────────────────────┬──────────────────────┬─────────────────────┐
 │                        │                      │                     │
 │ Device Modes           │        default       │    min-width:1024   │
 │                        │                      │                     │
 ├────────────────────────┼──────────────────────┼─────────────────────┤
 │                        │                      │                     │
 │ spacing/lg             │ spacing/10           │ spacing/16          │
 │                        │                      │                     │
 │ ...                    │ ...                  │ ...                 │
 │                        │                      │                     │
 └────────────────────────┴──────────────────────┴─────────────────────┘
```

### 2. 响应式字体大小

为不同屏幕尺寸设置合适的字体大小：

```
 ┌────────────────────────┬──────────────────────┬─────────────────────┐
 │                        │                      │                     │
 │ Device Modes           │        default       │    min-width:1024   │
 │                        │                      │                     │
 ├────────────────────────┼──────────────────────┼─────────────────────┤
 │                        │                      │                     │
 │ font/hero/size         │ font/2xl/size        │ font/3xl/size       │
 │                        │                      │                     │
 │ ...                    │ ...                  │ ...                 │
 │                        │                      │                     │
 └────────────────────────┴──────────────────────┴─────────────────────┘
```
