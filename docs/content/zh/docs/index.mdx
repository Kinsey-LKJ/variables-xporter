# Variables Xporter

Variables Xporter 是一个专业的 Figma 插件，支持将 Figma Variables 导出为 **shadcn/ui 主题变量**、**Tailwind CSS 配置文件**或 **CSS Variables**，让 Design Tokens 轻松同步到开发环境。

## shadcn/ui 主题变量支持

<Callout type="info" emoji="✨">
Variables Xporter 支持将 Figma Variables 直接导出为 shadcn/ui 主题变量，让你的 shadcn/ui 组件拥有完整的主题定制能力。
</Callout>

了解如何将 Figma Variables 导出为 shadcn/ui 主题配置，支持明暗模式、品牌颜色模式、密度模式等：
<Cards.Card
  icon={<Star />}
  title="shadcn/ui 主题导出"
  href="/docs/shadcn-ui"
  className="mt-4"
/>

## 导出模式

- **shadcn/ui (Tailwind CSS V4)** ⭐: 生成 shadcn/ui 主题变量和 Tailwind CSS V4 配置
- **shadcn/ui (Tailwind CSS V3)** ⭐: 生成 shadcn/ui 主题变量和 Tailwind CSS V3 配置
- **Tailwind CSS V4**: 生成 Tailwind CSS V4 版本的配置文件和对应的 CSS
- **Tailwind CSS V3**: 生成 Tailwind CSS V3 版本的配置文件和对应的 CSS
- **CSS Variables**: 生成标准 CSS Variables
