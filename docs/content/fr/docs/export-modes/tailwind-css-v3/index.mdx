# Tailwind CSS V3

Lors de l'exportation avec Tailwind CSS, Variables Xporter génère deux fichiers :

- `tailwind.config.js` - Le fichier de configuration de Tailwind CSS
- `global.css` - Contenant toutes les définitions de variables CSS

## Caractéristiques et optimisations

### 1. Optimisation des valeurs de couleur

Conversion automatique des valeurs de couleur au format de canaux RGB, prenant en charge les modificateurs d'opacité de Tailwind :

```js
// tailwind.config.js
colors: {
  primary: 'rgb(var(--colors-primary-default))'
}

// global.css
:root {
  --colors-primary-default: 192 57 57;  /* Format de canaux RGB */
}
```

Exemple d'utilisation :

```html
<div class="text-primary/80">
  <!-- 80% d'opacité -->
  Bonjour le monde
</div>
```

### 2. Fusion des variables de police

Fusion automatique des variables de police associées, conformément aux conventions de Tailwind :

```js
fontSize: {
  'heading-1': ['2.5rem', { lineHeight: '1.2' }]
}
```

### 3. Prise en charge multi-mode

Grâce au **traçage des variables** et au **principe de point d'entrée unique**, Variables Xporter peut réaliser une exportation complexe de variables multi-modes.

```css
:root {
  --colors-primary: var(--colors-primary-light);
}

.dark {
  --colors-primary: var(--colors-primary-dark);
}
```

### 4. Compatibilité puissante

Variables Xporter prend en charge les méthodes d'exportation de Tailwind CSS V3 et Tailwind CSS V4, et un ensemble de variables Figma peut être basculé sans problème entre les deux méthodes d'exportation sans nécessiter de modifications. Variables Xporter ajuste automatiquement l'exportation en interne selon les conventions de nommage des différentes versions.

## Recommandations pour l'organisation des variables

Pour garantir une exportation parfaite des variables Figma, la dénomination des variables doit suivre certaines règles, car elles doivent correspondre aux clés de configuration de Tailwind CSS. De plus, une bonne organisation des variables offre de nombreux avantages :

- Améliore la maintenabilité des variables
- Maintient la cohérence du système de design
- Facilite la collaboration en équipe
