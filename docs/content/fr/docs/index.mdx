# Variables xporter

Variables xporter est un plugin Figma qui permet d'exporter les variables Figma vers un fichier de configuration Tailwind CSS ou des variables CSS. Synchronisez facilement votre système de design avec votre environnement de développement !

## Formats d'exportation

- **Tailwind CSS** : génère un fichier de configuration et le fichier CSS correspondant
- **Variables CSS** : génère des variables CSS standard

## Recommandations pour l'organisation des variables Figma

Nous fournissons des recommandations complètes pour l'organisation des variables Figma, vous aidant à mieux gérer vos variables et à garantir l'utilisabilité des résultats exportés.
