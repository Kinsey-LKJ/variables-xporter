# Variables Xporter

Variables Xporter est un plugin Figma professionnel qui transforme les Variables Figma en **variables de thème shadcn/ui**, **fichiers de configuration Tailwind CSS** ou **Variables CSS** prêts pour la production. Rationalisez votre flux de travail design-vers-développement et maintenez la cohérence de vos Design Tokens dans votre projet.

## Pourquoi Choisir Variables Xporter ?

<Callout type="info" emoji="✨">
Un plugin qui exporte directement les Variables Figma en tant que configurations de thème shadcn/ui, permettant une personnalisation complète de vos composants shadcn/ui avec un setup minimal requis.
</Callout>

Apprenez comment exporter les Variables Figma en tant que configurations de thème shadcn/ui, supportant les modes clair/sombre, modes de couleur de marque, modes de densité, et plus :

## Modes d'Exportation

- **shadcn/ui (Tailwind CSS V4)** ⭐: Génère les variables de thème shadcn/ui et la configuration Tailwind CSS V4
- **shadcn/ui (Tailwind CSS V3)** ⭐: Génère les variables de thème shadcn/ui et la configuration Tailwind CSS V3
- **Tailwind CSS V4**: Génère les fichiers de configuration Tailwind CSS V4 et le CSS correspondant
- **Tailwind CSS V3**: Génère les fichiers de configuration Tailwind CSS V3 et le CSS correspondant
- **Variables CSS**: Génère les Variables CSS standard
