# 功能特性 ✨

让我们一起探索 Variables Xporter 提供的强大功能，这些功能可以帮助你轻松地将 Figma 变量导出到开发环境。无论你是设计师还是开发者，都能找到适合自己的使用方式。

## 核心功能

<Cards>
  <Cards.Card
    icon={<ArrowsUpDown />}
    title="变量导出"
    href="/docs/features/variable-export"
  >
    支持将 Figma 变量导出为 Tailwind CSS 配置或 CSS 变量，保持设计系统的一致性。
  </Cards.Card>
  <Cards.Card
    icon={<Type />}
    title="单位转换"
    href="/docs/features/unit-conversion"
  >
    自动将 px 单位转换为 rem，支持精确的小数位控制。
  </Cards.Card>
  <Cards.Card
    icon={<Monitor />}
    title="多模式支持"
    href="/docs/features/multi-mode"
  >
    完整支持明暗模式和多主题系统，轻松实现主题切换。
  </Cards.Card>
  <Cards.Card
    icon={<Layers />}
    title="变量引用处理"
    href="/docs/features/variable-references"
  >
    智能处理变量间的引用关系，确保导出结果的正确性。
  </Cards.Card>
</Cards>
