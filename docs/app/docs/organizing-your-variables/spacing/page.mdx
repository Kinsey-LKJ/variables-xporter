# 间距和尺寸

用于尺寸和间距样式变量的组织建议。


## 顶层目录
一般情况下，建议直接以 `spacing` 作为顶层目录，例如这样命名：

- `spacing/sm`
- `spacing/md`
- `spacing/lg`
- `...`

对于原子化变量：
- `spacing/0`
- `spacing/1`
- `spacing/2`
- `...`

<Callout type="error" emoji="🚨">
在用于 Tailwind CSS 项目时，将 `spacing` 作为顶层目录是必须的。
</Callout>

## 单位

Variables Xporter 会自动处理单位转换，你可以在 Figma 中使用 px 单位，并在导出时勾选 "使用 Rem 单位" 选项。
