# Tailwind CSS

选择 Tailwind CSS 导出方式时，Variables Xporter 会生成两个文件：
- `tailwind.config.js` - Tailwind CSS 的配置文件
- `global.css` - 包含所有 CSS 变量定义

## 特性和优化

### 1. 颜色值优化
自动将颜色值转换为 RGB 通道格式，支持 Tailwind 的不透明度修饰符：

```js
// tailwind.config.js
colors: {
  primary: 'rgb(var(--colors-primary-default))'
}

// global.css
:root {
  --colors-primary-default: 192 57 57;  /* RGB 通道格式 */
}
```

使用示例：
```html
<div class="text-primary/80">  <!-- 80% 不透明度 -->
  Hello World
</div>
```

### 2. 字体变量合并
自动合并相关的字体变量，符合 Tailwind 的约定：

```js
fontSize: {
  'heading-1': ['2.5rem', { lineHeight: '1.2' }]
}
```

### 3. 多模式支持
自动生成完整的多维变量系统配置，让你的设计系统更加灵活和强大：

```js
// tailwind.config.js
module.exports = {
  darkMode: 'class',
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: 'rgb(var(--colors-primary-light))',
          dark: 'rgb(var(--colors-primary-dark))',
          'brand-zhiyun': 'rgb(var(--colors-zhiyun-primary))',
          'brand-fiveray': 'rgb(var(--colors-fiveray-primary))',
        }
      },
      spacing: {
        container: {
          DEFAULT: 'var(--spacing-container-default)',
          md: 'var(--spacing-container-md)',
          lg: 'var(--spacing-container-lg)',
        }
      },
      fontSize: {
        base: {
          DEFAULT: ['var(--font-size-base-default)', { lineHeight: 'var(--line-height-base)' }],
          md: ['var(--font-size-base-md)', { lineHeight: 'var(--line-height-base)' }],
          lg: ['var(--font-size-base-lg)', { lineHeight: 'var(--line-height-base)' }],
          compact: ['var(--font-size-base-compact)', { lineHeight: 'var(--line-height-base-compact)' }],
        }
      }
    }
  }
}

/* global.css */
:root {
  --colors-primary-light: 229 85 23;      /* #E55517 */
  --colors-primary-dark: 238 109 33;      /* #EE6D21 */
  --colors-zhiyun-primary: 229 85 23;     /* #E55517 */
  --colors-fiveray-primary: 218 89 89;    /* #DA5959 */
}
```

通过[多模式变量管理](/docs/organizing-your-variables/multi-mode)了解如何组织和管理复杂的多模式变量系统，以及查看[多模式支持特性](/docs/features/multi-mode)了解更多使用细节。Variables Xporter 不仅能自动处理变量转换，还能帮你构建一个支持主题、品牌、响应式、布局密度等多个维度的完整设计系统。

## 必须遵守的规则

### 1. 顶层结构
变量集合必须使用 Tailwind CSS 的主题配置名：

- `colors` - 颜色变量
- `spacing` - 间距、宽度、高度等
- `fontSize` - 字体大小
- `fontWeight` - 字重
- `borderRadius` - 圆角
- `opacity` - 透明度
等。

### 2. 变量层级
使用 `DEFAULT` 作为基础值的标记：

```
colors/
  primary/
    DEFAULT     -> text-primary
    foreground  -> text-primary-foreground
```
