# Tailwind CSS

选择 Tailwind CSS 导出方式时，Variables Xporter 会生成两个文件：
- `tailwind.config.js` - Tailwind CSS 的配置文件
- `global.css` - 包含所有 CSS 变量定义

## 配置文件导出

<Steps>
### 基础配置
生成标准的 Tailwind 配置文件：

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: 'rgb(var(--colors-primary-default))'
      },
      fontSize: {
        'heading-1': ['2.5rem', { lineHeight: '1.2' }]
      }
    }
  }
}
```

### 颜色值优化
自动将颜色值转换为 RGB 通道格式，支持 Tailwind 的不透明度修饰符：

```css
/* global.css */
:root {
  --colors-primary-default: 192 57 57;  /* RGB 通道格式 */
}
```

### 变量合并
自动合并相关的变量，符合 Tailwind 的约定：

```js
fontSize: {
  'heading-1': ['2.5rem', {
    lineHeight: '1.2',
    letterSpacing: '-0.02em',
    fontWeight: '600'
  }]
}
```
</Steps>

## 自定义选项

### 变量映射

<Callout type="info">
  你可以自定义变量如何映射到 Tailwind 的配置结构中。
</Callout>

```js
module.exports = {
  theme: {
    extend: {
      // 自定义映射
      colors: {
        'brand': 'rgb(var(--colors-primary-default))',
        'brand-accent': 'rgb(var(--colors-secondary-default))'
      }
    }
  }
}
```

### 主题模式

支持多主题模式的配置：

```css
/* global.css */
:root {
  --colors-primary: var(--colors-primary-light);
}

.dark {
  --colors-primary: var(--colors-primary-dark);
}
```

### 高级配置

<Steps>
### 预设支持
可以与 Tailwind 预设（preset）结合使用：

```js
// tailwind.config.js
module.exports = {
  presets: [require('./preset.js')],
  theme: {
    extend: {
      // 你的主题配置
    }
  }
}
```

### 插件集成
支持与其他 Tailwind 插件协同工作：

```js
module.exports = {
  plugins: [
    require('@tailwindcss/forms'),
    require('@tailwindcss/typography')
  ]
}
```

### 自定义变量前缀
可以配置 CSS 变量的前缀：

```js
module.exports = {
  prefix: 'tw-'  // 添加前缀
}
```
</Steps>
