# CSS 变量

选择 CSS Variables 导出方式时，Variables Xporter 会生成一个包含所有 CSS 变量定义的 CSS 文件。这种方式更加简单和灵活，适合不使用 Tailwind CSS 的项目。

## 基础用法

<Steps>
### 变量定义
生成标准的 CSS 自定义属性：

```css
:root {
  --colors-primary: #c03939;
  --colors-primary-foreground: #ffffff;
  --font-size-heading-1: 2.5rem;
  --line-height-heading-1: 1.2;
}
```

### 变量使用
在任何 CSS 规则中使用这些变量：

```css
.button {
  background-color: var(--colors-primary);
  color: var(--colors-primary-foreground);
  font-size: var(--font-size-heading-1);
  line-height: var(--line-height-heading-1);
}
```

### 主题切换
使用 CSS 类切换主题：

```css
:root {
  --colors-primary: var(--colors-primary-light);
}

.dark {
  --colors-primary: var(--colors-primary-dark);
}
```
</Steps>

## 高级配置

### 媒体查询支持

<Callout type="info">
  支持基于媒体查询的响应式设计。
</Callout>

```css
@media (min-width: 768px) {
  :root {
    --spacing-container: 2rem;
    --font-size-heading-1: 3rem;
  }
}
```

### 变量计算

支持 CSS 计算函数：

```css
:root {
  --spacing-base: 1rem;
  --spacing-large: calc(var(--spacing-base) * 2);
  --spacing-small: calc(var(--spacing-base) * 0.5);
}
```

### 嵌套主题

支持多层主题嵌套：

```css
.theme-blue {
  --primary: var(--blue-500);
  --secondary: var(--blue-300);
}

.theme-red {
  --primary: var(--red-500);
  --secondary: var(--red-300);
}
```
