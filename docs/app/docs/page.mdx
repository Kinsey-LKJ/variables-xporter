# Variables Xporter

将设计系统无缝同步到开发环境！Variables Xporter 让设计师和开发者之间的协作更加顺畅，通过自动化导出 Figma 变量，消除手动同步的痛点。

Seamlessly sync your design system to development! Variables Xporter bridges the gap between designers and developers by automating the export of Figma variables, eliminating manual synchronization pain points.

## 为什么选择 Variables Xporter？ | Why Variables Xporter?

- 🚀 **效率提升** | **Boost Efficiency**
  从手动复制粘贴到一键导出，节省宝贵的开发时间
  From manual copy-paste to one-click export, save precious development time

- 🎯 **精确同步** | **Precise Synchronization**
  确保设计变量完全匹配，不会出现细微的差异
  Ensure design variables match exactly, eliminating subtle discrepancies

- 🔄 **实时更新** | **Real-time Updates**
  随时导出最新的设计变量，保持设计系统的同步
  Export the latest design variables anytime, keeping your design system in sync

## 核心功能 | Core Features

### 无缝集成 | Seamless Integration

- **Tailwind CSS**
  完美集成到你的 Tailwind 项目中，生成即可使用的配置文件
  Perfect integration with your Tailwind projects, generating ready-to-use configuration files

- **CSS 变量** | **CSS Variables**
  生成标准的 CSS 变量，适用于任何项目
  Generate standard CSS variables that work with any project

### 智能变量处理 | Smart Variable Handling

- **类型转换** | **Type Conversion**
  自动处理颜色、数值、尺寸等变量类型
  Automatically handle colors, numbers, dimensions, and more

### 多模式系统 | Multi-mode System

- **统一管理** | **Unified Management**
  基于单一出口原则，轻松管理多个主题和品牌模式
  Easily manage multiple themes and brand modes with a single source of truth

- **智能解析** | **Smart Resolution**
  自动处理不同模式下的变量值和引用关系
  Automatically handle variable values and references across different modes

- **代码优化** | **Code Optimization**
  生成简洁、规范、易于维护的配置代码
  Generate clean, standardized, and maintainable configuration code
