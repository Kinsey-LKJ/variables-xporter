# Variables Xporter

Variables Xporter 是一个 Figma 插件，帮助设计师和开发者更好地协作。通过导出 Figma 变量为标准格式，使设计系统更容易同步到开发环境中。

## 为什么选择 Variables Xporter？

- 🔧 **简化工作流**
  通过导出功能辅助设计变量的同步，提高工作效率

- 📦 **标准化输出**
  支持导出为 Tailwind CSS 配置和标准 CSS 变量

- 🎨 **灵活配置**
  提供多种配置选项，满足不同项目需求

## 核心功能

### 无缝集成

- **Tailwind CSS**
  完美集成到你的 Tailwind 项目中，生成即可使用的配置文件

- **CSS 变量**
  生成标准的 CSS 变量，适用于任何项目

### 智能变量处理

- **类型转换**
  自动处理颜色、数值、尺寸等变量类型

### 多模式系统

- **统一管理**
  基于单一出口原则，轻松管理多个主题和品牌模式

- **智能解析**
  自动处理不同模式下的变量值和引用关系

- **代码优化**
  生成简洁、规范、易于维护的配置代码
