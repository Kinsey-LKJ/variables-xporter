# Variables Xporter 🎨

Variables Xporter 是一个强大的 Figma 插件，让设计师和开发者之间的协作更加顺畅。它能智能地将 Figma 设计变量转换为开发代码，支持 Tailwind CSS 和原生 CSS 变量，确保设计系统在整个产品中保持完美的一致性。✨

## 开始使用 🚀

<Steps>
1. **安装插件** - 在 Figma 插件市场安装 Variables Xporter
2. **组织变量** - 按照[最佳实践](/docs/organizing-your-variables)组织你的设计变量
3. **选择模式** - 选择 [Tailwind CSS](/docs/export-modes/tailwind) 或 [CSS 变量](/docs/export-modes/css-variables)
4. **导出代码** - 一键导出，即刻使用
</Steps>
