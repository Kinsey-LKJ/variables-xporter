# Variables Xporter

将设计系统无缝同步到开发环境！Variables Xporter 让设计师和开发者之间的协作更加顺畅，通过自动化导出 Figma 变量，消除手动同步的痛点。

## 为什么选择 Variables Xporter？

- 🚀 **效率提升**
  从手动复制粘贴到一键导出，节省宝贵的开发时间

- 🎯 **精确同步**
  确保设计变量完全匹配，不会出现细微的差异

- 🔄 **实时更新**
  随时导出最新的设计变量，保持设计系统的同步

## 核心功能

### 无缝集成

- **Tailwind CSS**
  完美集成到你的 Tailwind 项目中，生成即可使用的配置文件

- **CSS 变量**
  生成标准的 CSS 变量，适用于任何项目

### 智能变量处理

- **类型转换**
  自动处理颜色、数值、尺寸等变量类型

### 多模式系统

- **统一管理**
  基于单一出口原则，轻松管理多个主题和品牌模式

- **智能解析**
  自动处理不同模式下的变量值和引用关系

- **代码优化**
  生成简洁、规范、易于维护的配置代码
