# Variables Xporter

Variables Xporter 是一个革新性的 Figma 插件，它彻底改变了设计系统的构建方式。不仅仅是简单的变量导出工具，它更是一个完整的设计系统解决方案，帮助你打造真正灵活、可扩展的设计体系。

## 为什么选择 Variables Xporter？

### 🎨 设计系统自动化
从简单的颜色变量到复杂的多维设计系统，Variables Xporter 都能优雅地处理。它让你专注于创造优秀的设计，而不是在设计和开发之间来回切换。

### 🔄 无缝的工作流
- **智能变量导出** - 自动转换为 Tailwind CSS 配置或 CSS 变量
- **实时同步更新** - 设计变更立即反映到代码中
- **多模式支持** - 轻松处理主题、品牌、响应式等多个维度
- **智能类型转换** - 自动处理颜色格式、单位换算等

### 💪 强大而灵活
构建一个真正强大的设计系统从未如此简单：
- 打造完整的多维变量体系
- 建立清晰的变量继承关系
- 实现模块化的设计管理
- 确保设计与代码的一致性

### 🚀 提升团队效率
- 设计师专注于创造，无需担心技术细节
- 开发者获得完美适配的代码配置
- 产品团队享受快速迭代的便利
- 各角色协作更加顺畅高效

## 功能特点

### 导出格式
- **Tailwind CSS**：生成配置文件和对应的 CSS 文件
- **CSS 变量**：生成标准 CSS 变量

### 变量处理
- **类型支持**：颜色、数值、尺寸等变量类型的自动转换
- **单位转换**：px 到 rem 的自动转换，保持精度
- **变量引用**：完整支持变量间的引用关系
- **变量过滤**：支持选择性导出，避免冗余配置

### 多模式系统
- **单一出口**：基于单一出口原则的多模式变量管理
- **智能解析**：自动处理不同模式下的变量值和引用
- **类型检测**：智能识别和转换不同类型的变量
- **代码优化**：生成简洁规范的配置代码
