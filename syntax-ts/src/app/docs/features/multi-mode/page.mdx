# 多模式支持

多模式支持是 Variables Xporter 的一个强大特性，它让你能够轻松导出多模式的设计变量，如明暗主题、品牌主题等。

## 什么是多模式？

多模式允许一个变量在不同情况下拥有不同的值。了解常见的模式包括：

- 明暗主题
- 品牌主题
- 响应式断点
- 布局松散度

Variables Xporter **支持复合的多模式变量**，例如 `colors/primary/DEFAULT`（主要颜色）变量可能同时受到明暗主题以及品牌主题的影响，这会导出以下结果：

```css
/* Default Mode */
:root {
  --colors-brand-a-500: 218 89 89 / 1;
  --colors-brand-a-600: 192 57 57 / 1;
  --colors-brand-b-500: 238 109 33 / 1;
  --colors-brand-b-600: 229 85 23 / 1;
  --colors-primary-default: var(--colors-primary-default-theme-modes);
  --colors-primary-default-theme-modes: var(--colors-primary-light-brand-modes);
  --colors-primary-light-brand-modes: var(--colors-brand-a-600);
  --colors-primary-dark-brand-modes: var(--colors-brand-a-500);
}

/* Mode Override */
.dark {
  --colors-primary-default-theme-modes: var(--colors-primary-dark-brand-modes);
}

/* Mode Override */
.brand-b {
  --colors-primary-light-brand-modes: var(--colors-brand-b-600);
  --colors-primary-dark-brand-modes: var(--colors-brand-b-500);
}
```
现在，`colors/primary/DEFAULT` 变量在不同的明暗主题以及品牌主题下，都会呈现不一样的结果。了解更多关于[多模式变量的组织建议](/docs/organizing-your-variables/multi-mode)。


## 工作原理
Variables Xporter 会追溯所选的导出集合中所有变量，直到找到变量的原始值。这意味着你可以跨集合引用变量，Variables Xporter 会完整的还原变量的引用链。
